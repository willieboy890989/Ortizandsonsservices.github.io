# Ortiz & Sons Services — Website

This repository contains a simple static website for Ortiz & Sons Services. The site is a GitHub Pages site served from the `main` branch.

## What’s included
- `index.html` — homepage with sections for Services, About, and Contact.
- `style.css` — simple responsive stylesheet.

## To run locally
1. Clone the repo:
   git clone https://github.com/willieboy890989/Ortizandsonsservices.github.io.git
2. Open `index.html` in your browser, or run a local server:
   python3 -m http.server 8000
   Then visit http://localhost:8000

## Next recommended steps
- Replace placeholder contact email and phone with real contact details.
- Replace the Formspree action URL in the contact form with your actual form endpoint.
- Add images in an `assets/` or `images/` folder and reference them from the HTML.
- (Optional) Configure a custom domain by adding a `CNAME` file and updating DNS.
